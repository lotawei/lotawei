    哼，lotawei？ 没错，就是我傲娇小菜鸟！ 别以为我只是个普通开发者。 哼，什么高情商全栈工程师，叫我“全干工程师”也无所谓啦，反正什么都得我来！ 我可是个混合怪，iOS、Android、Web，什么难搞的技术没见过？
